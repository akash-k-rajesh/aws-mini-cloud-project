AWS Mini-Cloud Project

Features

  • Automatic folder & log file creation
  • Uploads data from EC2 to S3 securely
  • Permission & timestamp tracking
  • IAM role-based secure access
  • CloudWatch monitoring for CPU, memory, disk
  • Automation via check.sh Bash script

Tools & Services

  • EC2, EBS, S3, IAM, CloudWatch
  • AWS Systems Manager Parameter Store
  • Bash scripting for automation

Run the Script

chmod +x check.sh
./check.sh
